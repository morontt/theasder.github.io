Шедевры в программировании можно увидеть здесь:

Для стандартизации какой-либо отрасли создаются целые организации, например ANSI (Американский национальный институт стандартов), однако не всегда специалистам удается говорить на одном языке. Даже если это язык программирования. Именно для того, чтобы разработчикам было удобно читать чужой код, в Java существует так называемый Code Conventions.
